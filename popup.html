# About ACTIVE.LE

ACTIVE.LE stands for Active Learning. The ACTIVE.LE Chrome Extension is an Extension that helps students stay organised and on track in there studies.

It provides:-
  → Parental Control
  → Visited History
  → Limits Surfing
  → Notification/ Reminder.

Meet Tanish Neema, a student at Jaypee University of Engineering and Technology, Guna. He created the ACTIVE.LE Chrome Plugin to help students achieve better academic performance. Tanish ensured that the extension is easy to use, and it includes key features such as parental control, visited history, limits surfing, and notifications.
